# 路由管理指南

## 概述

cc-admin 基于 Vue Router 4+ 构建了完整的路由管理系统，采用模块化设计，支持路由懒加载、权限控制、路由守卫等企业级功能。

## 🏗️ 架构设计

### 目录结构

```
src/router/
├── index.ts              # 🚪 路由器配置和导出
├── modules/              # 📦 路由模块
│   ├── core.ts           # 核心路由（登录、404等）
│   ├── dashboard.ts      # 仪表盘路由
│   ├── error.ts          # 错误页面路由
│   └── example.ts        # 示例页面路由
└── utils/                # 🔧 路由工具
    ├── customs.ts        # 自定义路由处理
    ├── helper.ts         # 路由辅助函数
    └── index.ts          # 工具函数导出
```
