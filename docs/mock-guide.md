# Mock 数据指南

## 概述

CC-Admin 使用 `vite-plugin-mock` 提供 Mock 数据服务，支持开发环境下的接口模拟，无需依赖后端服务即可进行前端开发。

## 🏗️ 目录结构

```
src/mock/
├── modules/              # Mock 模块
│   ├── auth.ts           # 认证相关 Mock
│   └── router.ts         # 路由相关 Mock
```
