# 构建配置指南

## 概述

CC-Admin 基于 Vite 7+ 构建现代化的前端工程化体系，提供高性能的开发体验和优化的生产构建。

## 🏗️ 构建架构

### 技术栈

| 技术           | 版本  | 作用                 |
| -------------- | ----- | -------------------- |
| **Vite**       | 7+    | 构建工具和开发服务器 |
| **TypeScript** | 5+    | 类型检查和编译       |
| **UnoCSS**     | 0.66+ | 原子化CSS编译        |
| **ESLint**     | 9+    | 代码质量检查         |
| **Prettier**   | 3+    | 代码格式化           |

### 配置文件

- `vite.config.ts` - Vite 主配置文件
- `tsconfig.json` - TypeScript 配置
- `uno.config.ts` - UnoCSS 配置
- `eslint.config.ts` - ESLint 配置
